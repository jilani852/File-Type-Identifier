# File-Type-Identifier

Hey friend!
I've created a program that lets you identify the type of a file. To run this program, simply do this:
  1-create a folder
  2-put any type of file you want next to file_idetifier.py
  3-open your command line in the same folder 
  4-run this command python file_identifier.py <file_path> or py file_identifier.py <file_path>
